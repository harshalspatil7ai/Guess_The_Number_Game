📌 Project Title

Guess The Number Game

📖 Description

This is a simple Python-based number guessing game where the computer generates a random number between 1 and 100, and the user tries to guess it.

The program provides hints like "Higher" or "Lower" after each guess until the correct number is found.

🚀 Features
🎯 Random number generation
📈 Hints for user (Higher / Lower)
🔢 Tracks number of attempts
🎮 Interactive gameplay
🧠 Concepts Used
Python Basics
Loops (while)
Conditional Statements (if-else)
Random Module
🛠️ Tech Stack
Python
